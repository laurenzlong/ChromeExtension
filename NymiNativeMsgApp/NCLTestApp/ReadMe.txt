========================================================================
    CONSOLE APPLICATION : NCLTestApp Project Overview
========================================================================

Must add the registry key: (either using Regedit or an installer)
HKEY_LOCAL_MACHINE\SOFTWARE\Google\Chrome\NativeMessagingHosts\com.nymi.nativemsg
or HKEY_CURRENT_USER\SOFTWARE\Google\Chrome\NativeMessagingHosts\com.nymi.nativemsg, 

and set default value of that key to the full path to the manifest file.
